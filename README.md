# 2026 Peptide Price Dashboard

A simple, self-contained price lookup and order builder for the peptide-Anne-Dora product catalog.

## Features

- **Quick price lookup** — select any peptide and vial size to see kit price, per-vial cost, 10+ kit pricing, total quantity, and cost per mg
- **Volume savings indicator** — shows how much you save per mg when choosing larger vial sizes
- **Order builder** — add products with quantities, adjust inline with +/− buttons, running total updates in real time
- **Copy order** — one click copies a clean text summary (qty, abbreviation, name, cost) ready to paste anywhere
- **10+ kit auto-pricing** — automatically applies bulk rate when quantity hits 10+
- **Dark mode** — follows your system preference

## Usage

Just open `index.html` in any browser. No server, no dependencies, no install needed.

### Host on GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main** branch, root folder
4. Your dashboard will be live at `https://yourusername.github.io/peptide-dashboard/`
